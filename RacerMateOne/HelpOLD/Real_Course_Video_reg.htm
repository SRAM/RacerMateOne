Real Course Video Registration

Real Course Videos ship with the original Real Course Video application, which contain the video registration [unlock] tool.  Due to the actual number of Real Course Video's on the market the initial release of RacerMate One uses the registration routine supplied with the original software.  This will mean running the original software one time after installing a video to activate the software.  RacerMate One can determine whether a video is activated.

Registering Interactive Real Course Videos

Registration Instructions - Computer connected to the Internet

  1. Install the Interactive Real Course Video (IRCV) application and Video Course.
  2. Launch the IRCV application supplied with the video and go to Start - Video and choose the video you want to register.
  3. If the video is not registered, you will be prompted to register. Choose Yes to register.
  4. You will then be asked for your First Name, Last Name, and e-mail address. These are all required fields. When done, click OK.
  5. If your Internet connection was successful to communicate with our registration server, there should be a "Registration Success" message. Click OK to close that message and the video will open in a registered form.  You can now close the original IRCV application to use this video in RacerMate One.
If the registration wasn't successful: If an error occurs, you can try to register again as normal, or attempt a manual registration as shown below.

Registration Instructions - Remote Internet computer

The following registration routine was designed for those who do not have their CompuTrainer systems connected to the Internet. If you use the following steps and you are connected to the Internet on your CompuTrainer computer, just ignore the steps referring to copying and moving the reg files between computers.

During the installation of your video course a registration key was created in the C:\RCVReg folder on your computer. For instance; if you installed Ironman™ Coeur d'Alene, the reg file for that course was CDA.reg. To register manually you will need to transfer this *.reg file from your training software computer to your Internet-compatible computer and then upload it to the RacerMate registration server.  We have found the best way to transfer files from computer to computers is with a USB pocket drive. They are not very expensive and will aid in moving updates of your training software to the computer as well.  In this case you only need to get the "reg file" off your training software computer and have it available on your Internet computer to get your IRCV registration [unlock] key.

  1. Launch Windows Explorer or My Computer and browse to the C:\RCVReg folder. Open that folder and locate the reg file for the video you purchased. Copy the reg file to your pocket drive, floppy, or burn it to a CD-ROM.
  2. Go to the computer you have connected to the Internet.
  3. Go to the following web page: www.racermate.net/RCVreg.php
  4. On this page you will be asked to enter your name and e-mail address. Then "upload" the Reg file to the registration server by browsing to the drive containing the Reg file.
  5. You will then be provided a very long registration key.  You will need to copy this key over to the CompuTrainer computer. Best option is to highlight and copy the string of text and paste it into a simple text file (you can use Notepad.exe for this) and name the file something like reg.txt.  If you feel confident in copying the text down by hand and hand-typing it later, you can also do this.
  6. If you created a text file, copy it onto your transport pocket drive, or disc, and head back to the CompuTrainer computer.
  7. Browse your computer and locate the folder you chose to install the Real Course Video software (default is C:\Real Course Video). Within this folder locate vid.ini (the Windows description of this file will be "configuration settings").
  8. Open the file vid.ini using Notepad.exe and locate the section in this file called Registration.
  9. Copy the long key string from the reg.tx
