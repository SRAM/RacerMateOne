RacerMate One Demo Modes

Registering RacerMate One allows you to use it with your RacerMate training hardware.  When unregistered, the software is limited to demo mode.

Demo mode is useful to determine the capacity of the software to run on the computer in question.  It is better to not waste a registration attempt on a computer that can't run the software satisfactorily. 

What you can and can't do in demo mode

You can:

  • enter all screens and play with the software. 
  • use it to create riders.
  • detect hardware and get the hardware setup and ready to go.
  • run all courses with computer generated "fake" data.  This is most helpful in 3D Cycling to determine how fast your computer is as you add more riders to the demo mix.

You can't:

  • ride your trainer while connected.

Demo Mode in 3D

Even in a registered version of RacerMate One you may want to run demo mode.  Here's how to do it:

  1. Load a course and riders as you normally would and press Start to enter the race screen.
  2. Press the PC key letter "D" to enter Demo mode.  A small letter "D" appears in the bottom corner of the screen.
  3. If you selected only one or two riders, you can run will those or add more as "demo riders" by pressing the PC key letter "A".  Each press will add a rider, up to to the maximum of 8 riders.
  4. Press the number keys 1-8 and each number you press you can adjust that riders speed using the left/right mouse key. This will be true even after the timer starts.
  5. Press the PC key letter "G" to start the timer.
  6. Press the PC number key to select a rider, then "F" to change their camera view.
  7. Press the PC key "V" to switch to split screen views.  While in split screen views you can still select a rider using 1-8 and use the "F" key to change the camera views, etc.
  8. Press the PC key "R" to reset the race.

Demo Mode in Real Course Video

To use demo mode in Real Course Video, do the following:

  1. Load a video course and one rider as you normally would and press Start to enter the race screen.
  2. Press the PC key letter "D" to enter Demo mode.
