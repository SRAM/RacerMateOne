Course Creator

Course Creator within RacerMate One differs greatly from previous versions of Courses Creator.  It contains many of the same functions, but presents courses in every imaginable view to aid in changing and/or creating them. 

Features

  • Profile viewer breaking the course into "legs" to allow you to edit any leg.
  • Import data containing distance and elevation data (GPX courses containing Latitude, Longitude and Elevation are accepted).
  • Edit existing courses.
  • Create new courses in Distance/Time and grade, watts, and %AT.
  • Insert sections into existing courses.
  • Copy large ranges or data and paste them into another part of the course, or entire course to make 2 or more intervals.
  • Delete sections from existing courses.
  • Undo allows you to back out of all changes made.
  • Add wind to existing courses
  • Save existing courses as a new course to edit - leaving the original untouched.

Edit Existing Course

  1. To Edit any existing course, or to create a new course, go to any application staging screen where you select courses (with the exception of Real Course Video, which has no editable courses).
  2. Select a course type from the dropdown list to populate the course list with all courses of that type or the type you want to create.
  3. Select a course to Edit or any course if your intent is to create a new course from scratch.
  4. Press the Edit button found below the course list.  This will launch the Course Creator application and load the selected course into the editor.  If you intend to create a new course rather than edit this one, proceed to the New Course section below.
  5. After the course loads you will see every leg of the course on the profile as well as the list view of the course.
  6. Scroll through the profile using the horizontal slider or the list view with the vertical slider to find the spot you want to edit.  If you click on the profile or the list that leg will highlight.
  7. Double-click on the profile to open the edit box as shown above. 
  8. Edit the course leg by either using the horizontal sliders or by typing directly into the text boxes.
  9. When finished editing a leg, press Accept or if you do not want to save your changes press Cancel to return to the edit screen.
  10. Continue editing as needed and press the Save button to save the course or save-as a new course by saving it with a different name.  If you save it with a different name the original course will not be changed.

Please Note:

  • There are limits to the length/duration of an individual legs in Course Creator that may not meet the length or duration found in an existing course.  If you exceed, for instance, 10560 feet for a leg (2 miles) on an existing course, you may need to break up one big leg into multiple smaller legs if this is the case
  • If you are wanting to add new legs or copy and paste new legs, see the instructions for this further down this help file.
  • Reversing, out and back, and other lap functions can be applied to the course from within the standard RacerMate One course profile viewer.
  • Due to the way RacerMate One "views" courses, after you have created or edited a course and saved it, you must close RacerMate One and relaunch it to "refresh" the course list to see your finished work in its complete profile view.

Very large courses - Please Wait

Some courses may take a very long to to both open and edit since we have to redraw the course with every edit.  Every course you load will show the above message as it loads, but you will be prompted with a different message first is the course exceeds 10,000 legs.  If you accept the message you can edit the course, but you will be prompted with each edit to wait (as long as is required to redraw the new change) with the above message.

Copy/Paste

To use the Copy/Paste tools:

  1. Click on a line within the list view and this will activate the Copy command. Click Copy, select where you want to insert the leg segment, then click Before or After, then Paste.
  2. To select multiple legs, click on a leg and then Shift-click on the last leg you want to copy, then click Copy to copy this group. Then select where you want to insert it, and select Before or After, then Paste.  Doing this can make a course from a one with few legs to one with repeating multiple legs, as shown above.
  3. To select random legs to create a new group (order will be maintained), click on a leg, then CTRL-click on selective legs, then click Copy to copy this group. Then select where you want to insert it, and selected Before or After, then Paste.

Insert

To insert a new leg, select a leg and then press Insert.  It will insert a new leg with the same values seen in the leg preceding it.  Edit by double-click the profile cell.

Undo

To undo any step you have made since the beginning, press the Undo button as many times to return to where you want to resume building.

Delete

If you simply want to delete a leg, select a leg in the Profile and then click Delete.

Clear

If you copy sections, you can either select others or press Clear is you want to make the Paste button go away.

Notes

You can enter notes to save them within the course.  They will only appear within the Course Creator tool, but will help you in the future to better define what the reason might have been for creating the course or workout.

Course Stats

There are course statistics displayed below the Notes section.

Creating New Courses

Creating a new course requires first getting into the Course Creator tool.  To do so follow the same steps as Edit above to select any course of any course type from within Power Training and Edit it.  From here, follow these steps:

  1. Click the New button to display the dialog shown above.
  2. Select the course type you want to create.
  3. This will create a one leg course with some defaults to begin creating a new course from.
  4. Edit the first leg to begin, Accept, and then press Insert to add a second leg. 
  5. At some point you can copy and paste a group of legs to make more legs and edit those.
  6. When you are satisfied with your course, or workout, press Save to save the course.

Import GPX Courses

You can import GPX courses as long as they match a compatible format.

  1. Click Import to open a Windows dialog where you can browse to select your GPX file.
  2. Because GPX files may have thousands of legs, you will be presented with the Please Wait message as it loads.  It may take a fair amount of time to edit legs too, so be patient with edits.
  3. When happy with your course, press Save to create the new course.

Notes:

  • By "Compatible format" this means he GPX data contains latitude, longitude, and elevation data.  If it doesn't, a course from it is impossible.
  • There is the possibility for GPX data to produce extreme grades and or nonsensical data.  If you edit the course and delete or edit extreme examples of grade, or sections containing no distance (Nam will show for leg distance), you should be able to create a nice course from GPX with minor post-import editing.
