Display Options

You can set what you see for data on the screen for the various applications in RacerMate One by selecting the Display option.  We've created default settings, but you can also setup your own presets to select from in the future.  Perhaps one day you don't want to see your speed or cadence while riding (some sort of blind test), you can create a preset called blind for any selected application and save it for selecting again at any time without needing to change anything on this page in the future.

Customizing each application's databar

Each application has a custom databar display - with the exception of Power Training, which will display all the data available based upon the grade or watts course selection. 

There are default settings created you can use - or you can create, and save, a variety of setting groups for each application as well.

To create a new databar group setting

  1. First select the Application you want to set a new group for.  This will display the original default group.
  2. Select or deselect items as needed.  With the first change you make the "Save as" button will appear.
  3. Click on Save as to display a dialog where you can name this group.  The group has a hidden association to the application showing in the Application window, so there is no need to worry giving the group an application specific name; like, My 3D blind test -- though you can name the group anything you choose.  When, for instance, you are in 3D Cycling, the display options you created for 3D Cycling will only appear when running 3D.

Deleting a databar group

  1. Select the Name drop-down menu to select the group you want to delete
  2. Click Delete

Using the custom databar group

  1. If you have created a custom databar group, you will see them displayed on the Setup staging screen for the application you choose.  Select it using the drop-down selector on that screen to make it the active databar.
