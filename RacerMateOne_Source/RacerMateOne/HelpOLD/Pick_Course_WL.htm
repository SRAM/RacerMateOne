Pick a Road Course - Grade style

Course selection has been greatly improved in RacerMate One.  Not only are we capable of filtering out courses that don't apply to an application, you can also sort the courses by name, and distance and total climbing (Alt/Elev).  This will help you narrow your search for the specific course or difficulty level.  You can also type into the Find command to search for a specific course.  Depending on the application you choose, you will have a variety of pre-loaded courses to choose from.

Course selection in 3D Cycling, SpinScan training, and Real Course Video

Some applications, like 3D Cycling (shown below), SpinScan, and Real Course Video, have pre-filtered courses types.  Therefore you will not be able to select the wrong type of course even though all courses now what the same "RMC" file extension.

Course selection in Power Training

Power Training can run any course type and has a slightly different course selection tool.  Course selection in Power Training, because it is not limited to a specific graphic requirement (showing a physical road that hopefully looks realistic), can run any course or test imaginable.  All courses have a profile - and that is what drives the Power Training application. 

Course selection starts first with a selection of the course style (in the course drop-down selector to the right of the Browse button).  This will filter the selection of available courses on that course type.

Course types

  • Grade courses are "windload-based" courses of either "time and grade" or "distance and grade."   These control the trainer and adjust load based up slope or also called % grade.  In other words, the faster you go the more wind you have to push out of the way.  Going faster is harder simply due to wind increase.
  • ERG courses are "wattage-based" and will be either "time and watts" or "distance and watts."   These follow a preset wattage and will adjust the trainer as needed to maintain the wattage you are set to at the moment.  ERG courses demand as steady a state of your cycling as possible to minimize the adjustments (electronically) to maintain the wattage you are setting.  If your pedal RPM is fluctuating by vast amounts, the drive to the trainer also has to try to keep up with you.  It may fail if you don't keep your speed and cadence stable.
  • %AT courses are also "wattage-based" and will be either "time and %AT" or "distance and %AT."  Like ERG courses these follow a preset wattage, but instead of using a preset wattage it uses what you set as your Anaerobic Threshold wattage and adjusts the load based upon percentages of that value.  Also like ERG courses you will want to maintain a very steady RPM.

Changing the list sort

By default the courses are shown alpha-numerically, but you can sort the list by how many laps have been pre-saved into the course, the course length, or total climbing or altitude gain as well. 

To change the list sort

  1. Click on a header (Laps, Miles/Kms, or Alt/Elev) and the course will sort using that selection and will bolden to show they are active. 
  2. To reverse sort, press the same header again.
  3. To return to alpha-numeric again, click a 3rd time to unbolden a selected header.

Notes:

  • The slider used in RacerMate One doesn't behave like a standard Windows-based slider... as much as we wanted it to.  You need to grab the slider and drag it up and down to view the list.  It does offer some right-click functions standard (see image below) and we will try to program in standard Windows-based slider functions in the future.
  • Courses may need to "populate" the list, which can take a couple minutes if you have a less-than stellar computer.  This is an "indexing process" needed to allow the sort and find functions to work.  If you are witnessing the slider button growing smaller as you open to a setup/staging page, indexing is in process.

Selecting a grade course

  1. If using Power Training select the drop-down menu above the course list and choose Course (grade).
  2. Sort the list as indicated above.
  3. Scroll through the courses and click on any course to select it and display the profile and course data.
  4. If you are satisfied with the selection, press Start to load the course.

Change Start and End points

RacerMate One takes course manipulation to another level.  You can now select start points, end points, reverse selections, add out and backs to a selection and save all your changes as a new course.  No longer do you need to create courses that loop to have multiple laps.  Just select the start and end points and set the number of laps.  You are done.

The following section shows how to change the start and end points only.  The complete details on manipulating and creating new courses from within existing courses can be found in the link show below.

Set a start and end point

  1. Select a course to load it into the profile viewer.
  2. Hover over the Start post with your mouse and left-click/hold/drag the Start post to the new start position.
  3. Hover over the End post and left-click/hold/drag the End post to the new end position.
  4. Pay attention to the distances and values in the balloon to help zero in on the correct position you want these set to.
  5. You will notice as soon as you drag either the Start or End post, the Save as button will appear.  You can opt to save this "edit" as a new course.
  6. You can select the Zoom tool to help zero in on exactly where you want to start or end.

Zoom into a shortened course

When you have set a new Start and End point, you can click on the Zoom tab above the course profile to zoom into the course.  You will be able to fine-tune the Start and End points from this new vantage point.

Editing a course further before riding it

As described above, you can edit a course quite a bit before riding it.  This is explained in detail on the Course Profile edit page.
