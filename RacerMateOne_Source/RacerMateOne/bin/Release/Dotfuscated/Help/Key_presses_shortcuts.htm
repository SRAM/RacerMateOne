Key press shortcuts

The follow are the available key presses and shortcuts for each application while riding.

CompuTrainer:

NOTE: RESET on controller keypad refers to Stand-Alone functions ONLY. 
Reset in RacerMate One is F5 or  [RESET] plus [F2].

Velotron:

 

3D Cycling

Controller/PC key

Function
   
F1 or G Start Timer; Pause Race
F2 or S Display SpinScan data for single riders or personal data for riders in a split screen display
F2 Press and Hold (bike #1 only) - Enable/Disable split screen on multiple live rider races.
Toggle split screen contents - rider controlled
F3 Select a Pacer to adjust (any rider assigned)
F4 or F Select Camera position (highlighted rider)
F5 or R Reset timer; back to start; back to setup screen
F6 or B Velotron Emergency Brake
Up / Down keys Velotron Gear shifting (Velotron Only)
Adjust Pacer if selected (CompuTrainer Only)
Fn plus up/+ or down/- Adjust Pacer if selected (Velotron Only)

Real Course Video

F1 or G Start Timer; Pause Race
F2 or S Display SpinScan data for single riders or personal data for riders in a split screen display
F2 Press and Hold - Enable split screen on multiple live rider races.
F3 Not used
F4 Not used
F5 or R End ride; Reset timer; Back to setup screen
F6 or B Velotron Emergency Brake
Up / Down keys Velotron Gear shifting (Velotron Only)
Fn plus Up/+ or Down/- Not used

Power Training

F1 or G Start Timer; Pause Race
F2 Not used
F2 Not used
F3 Not used
F4 Not used
F5 or R End ride; Reset timer; Back to setup screen
F6 or B Velotron Emergency Brake
Up / Down Velotron Gear shifting (Velotron Only)
Adjust manual courses (CompuTrainer Only)
Fn plus Up/+ or Down/- Adjust manual courses (Velotron Only)

SpinScan

F1 or G Start Timer; Pause Race
F2 or S Display SpinScan data for single riders or personal data for riders in a split screen display
F2 Alternate the SpinScan image large to small
F3 Not used
F4 Not used
F5 or R End ride; Reset timer; Back to setup screen
F6 or B Velotron Emergency Brake
Up / Down keys Velotron Gear shifting (Velotron Only)
Increase/Decrease Grade values in manual controlled course (CompuTrainer Only)
Fn plus Up/+ or Down/- Increase/Decrease Grade values in manual controlled course (Velotron Only)
